# Car Price Prediction

## Project Overview
This project is part of the CodeAlpha Data Science Internship.

## Objective
Predict the selling price of a car using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Algorithm
- Linear Regression

## How to Run
1. Install the libraries:
   pip install -r requirements.txt
2. Place `car_data.csv` in the project folder.
3. Run:
   python car_price_prediction.py
